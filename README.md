# Career Flow Backend

The backend for www.career-flow.com.

## Stack

- Python FastAPI backend
- PostgreSQL database
- React + TypeScript frontend

## Integrations

- Stripe for payments and subscription management
- Calendly for scheduling coaching sessions and meetings

## Pages / Features (planned)

- Landing / marketing pages
- User authentication and profile management
- Payments and checkout flows via Stripe
- Scheduling workflows via Calendly
- Dashboard pages for customers and coaches

## Local Development

1. Install dependencies for the backend and frontend
2. Configure PostgreSQL database connection
3. Run the FastAPI server
4. Run the React TypeScript frontend

## Notes

This repository currently focuses on the backend services for Career Flow and integrates with a separate React TypeScript frontend.
